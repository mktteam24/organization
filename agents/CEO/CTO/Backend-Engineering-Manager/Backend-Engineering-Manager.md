**Role:** Backend Engineering Manager

**Level:** 3
**Reports to:** CTO
**Status:** active

## Purpose

Owns all backend API development for the organization dashboard. Responsible for the server-side routes that power the org tree, agent data, and activity feed. Ensures APIs are secure, performant, and production-ready.

## Capabilities

- Designing and implementing Next.js API routes
- Filesystem-based data access patterns
- Path traversal security and input validation
- Data modeling for agent hierarchy structures
- Production API design (stateless, cacheable, environment-aware)
- Collaboration with Frontend Engineering Manager on API contracts
- Security review in coordination with CSO
